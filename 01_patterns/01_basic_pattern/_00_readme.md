## ⚙️ Key Concepts

* 🔁 **Nested loops**

  * outer → rows
  * inner → columns
* 🔢 **Patterns = counts**

  * stars / numbers per row
* 📐 **Shape recognition**

  * increasing
  * decreasing
  * constant

---

## 🧠 Pattern Building Strategy

1. 👀 Visualize output
2. 📏 Count rows & columns
3. 🔄 Convert into loops
4. ⚡ Dry run once
