## 📊 Difficulty Flow

```text id="g7p2mx"
Traversal → Search
        ↓
Pairs → Subarrays
        ↓
Two Pointer
        ↓
Prefix Sum → Kadane
```

---

## 🧠 Pattern Recognition

* Max/min subarray → Kadane
* Range queries → prefix sum
* Sorted array → binary search
* Rearrangement → two pointer

---

## 📌 Pro Insight

> Arrays are not about storage…
